<h1>Trybewarts</h1>

<b>Este projeto foi feito em conjunto com <a href="https://github.com/Andre-ARS">André Rodrigues Santos</a></b>

<h2>Descrição sobre o projeto</h2>

<i>Trybewarts</i> é um formulário com a temática do universo de <a href="https://pt.wikipedia.org/wiki/Harry_Potter">Harry Potter</a>, desenvolvido para a prática de conceitos de <b>Flexbox</b> do <b>CSS3</b>, além da utilização dos formulários do <b>HTML5</b>.

<h2>Tecnologias utilizadas</h2>

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

<h2>Como executar o projeto</h2>

O projeto pode ser executado através do arquivo 'index.html'.
